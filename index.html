    3. Perkembangan HTML
      HTML telah mengalami perkembangan yang signifikan sejak awal mula pada tahun 1991. Dari sederhana versi 1.0, HTML berkembang menjadi HTML5 pada tahun 2014. HTML5 membawa perubahan besar dengan penambahan fitur multimedia seperti audio dan video tags, elemen canvas untuk gambar, dan dukungan untuk animasi serta aplikasi web interaktif. Ini memungkinkan pengembangan halaman web yang lebih menarik dan responsif di berbagai perangkat. Perkembangan HTML masih berlanjut berkat kontribusi komunitas pengembang, memastikan bahwa HTML tetap relevan dan mendukung teknologi web terbaru.

  2. Heading
    Dalam HTML (Hypertext Markup Language) "heading" merujuk pada elemen-elemen yang digunakan untuk membuat judul atau tajuk dalam sebuah dokumen web. Heading berfungsi untuk memberikan struktur dan hierarki pada konten, sehingga memudahkan pembaca dan mesin pencari dalam memahami isi dokumen. Heading biasanya ditampilkan dalam bentuk teks yang lebih besar dan lebih menonjol daripada teks biasa. Ada enam tingkatan heading yang dapat digunakan dalam HTML, yang ditandai dengan elemen-elemen <h1> hingga <h6>.

  3. Paragraph
    Dalam HTML, elemen <p> digunakan untuk membuat paragraf atau blok teks yang terstruktur. Paragraf adalah bagian dari konten yang berisi sejumlah teks yang membentuk satu pikiran atau ide utama. Elemen <p> membantu mengatur dan memformat teks menjadi paragraf yang lebih mudah dibaca.
    Klik disini untuk melihat contoh penerapan dari Paragraf...

  4. Text Formatting
    Text formatting dalam HTML mengacu pada cara mengubah tampilan dan presentasi teks di halaman web. Ini mencakup pengaturan elemen seperti tebal (bold), miring (italic), garis bawah (underline), warna teks, ukuran teks, dan lainnya untuk memberikan gaya visual yang sesuai dengan tujuan dan estetika halaman. Ada beberapa jenis penggunaan dari text formatting, diataranya adalah bold, italic, underline dan masih banyak lagi

  5. List
    1. Pengertian List
      Dalam HTML, kita dapat menggunakan dua jenis elemen untuk membuat daftar: ordered list (daftar berurutan) dan unordered list (daftar tak berurutan). Keduanya digunakan untuk mengorganisir dan mengelompokkan item-item dalam suatu urutan tertentu.

    2. Ordered List
      Dalam HTML, kita dapat menggunakan elemen <p> untuk membuat teks paragraf. Selain itu, kita juga dapat menggunakan elemen <ol> untuk membuat daftar berurutan. Dalam daftar berurutan, setiap item memiliki nomor urut yang menunjukkan posisinya dalam daftar. Nomor ini biasanya berupa angka atau huruf. Daftar berurutan berguna ketika kita ingin menampilkan item dalam urutan tertentu, seperti langkah-langkah, peringkat, atau urutan kronologis.

    3. Unordered List
      Elemen <ul> digunakan dalam HTML untuk membuat daftar tak berurutan. Dalam daftar tak berurutan, setiap item biasanya ditampilkan dengan simbol bullet (titik atau tanda lainnya). Daftar ini digunakan ketika kita ingin menunjukkan kumpulan item yang tidak memiliki urutan spesifik. kita juga dapat menggunakan elemen <ol> untuk membuat daftar berurutan, di mana setiap item memiliki nomor urut yang menunjukkan posisinya dalam daftar.

  6. Table, Image, dan Hyperlink
    Sesuai dengan namanya, table biasa digunakan untuk membuat table, image digunakan untuk menyisipkan gambar, dan hyperlink digunakan untuk menyisipkan link pada website, pada pembuatan table, tag-tag yang dibutuhkan adalah table (sebagai wadah), thead untuk head pada table, th yang merupakan isi dari table, kemudian tbody sebagai perwakilan dari body table yang isinya terbagi menjadi td dan tr, tr sendiri berarti baris sementara td berarti kolom. Pada contoh ini, saya akan menggabungkan penerapan dari table, image, dan hyperlink secara sekaligus agar terlihat lebih simple

  7. HTML Form
    HTML form adalah elemen HTML yang digunakan untuk mengumpulkan input dari pengguna. Input ini kemudian dapat dikirim ke server untuk diproses. Form biasanya digunakan untuk keperluan seperti pendaftaran, pengisian data, survei, dan lain sebagainya

  8. Form Login
    Form login sendiri biasa digunakan sebagai pintu masuk user ketika ingin hendak masuk kedalam sebuah aplikasi maupun website, contohnya bisa kita lihat pada aplikasi facebook, instagram, tiktok dan lain-lain.
